Q15: How do closures interact with var, let, and const in loops?

With var, all closures in a loop share the same variable, so they capture the final value. With let, each iteration gets a new binding, so closures capture the correct value for each iteration.

var funcsVar = [];
let funcsLet = [];
for (var k = 0; k < 3; k++) {
    funcsVar.push(() => k);
}
for (let l = 0; l < 3; l++) {
    funcsLet.push(() => l);
}
console.log(funcsVar.map(fn => fn())); // [3, 3, 3]
console.log(funcsLet.map(fn => fn())); // [0, 1, 2]
