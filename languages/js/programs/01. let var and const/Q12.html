Q12: What happens if you use let/const in a for loop header?

When using let in a for loop header, a new binding is created for each iteration, so closures capture the correct value. With var, the same variable is used, so closures capture the final value after the loop ends.

for (let i = 0; i < 2; i++) {
    setTimeout(() => console.log('let i:', i), 0); // 0, 1
}
for (var j = 0; j < 2; j++) {
    setTimeout(() => console.log('var j:', j), 0); // 2, 2
}
