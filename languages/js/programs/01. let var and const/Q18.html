Q18: What is the difference between const object and const primitive?

const means the variable cannot be reassigned, but for objects and arrays, their contents can still be changed (mutated).

const arr = [1, 2];
arr.push(3); // allowed
// arr = [4, 5]; // Error

const obj = { a: 1 };
obj.b = 2; // allowed
// obj = {}; // Error
