Q13: Can you attach properties to variables declared with var, let, or const in the global scope?

In browsers, var declarations in the global scope become properties of the window object. let and const do not.

var globalVar = 1;
let globalLet = 2;
const globalConst = 3;
console.log(window.globalVar); // 1
// console.log(window.globalLet); // undefined
// console.log(window.globalConst); // undefined
