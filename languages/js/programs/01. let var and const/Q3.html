Q3: What is temporal dead zone (TDZ)?

The temporal dead zone (TDZ) is the period between entering scope and variable declaration for let and const. Accessing the variable in the TDZ throws a ReferenceError.

// console.log(y); // ReferenceError
let y = 10;
// console.log(z); // ReferenceError
const z = 15;
