Q5: Can you shadow a variable declared with var using let or const?

Yes, you can shadow a var variable with let or const in a block scope. The inner variable is only accessible within the block.

var num = 100;
{
    let num = 200; // allowed, shadows outer 'num'
    console.log(num); // 200
}
console.log(num); // 100
