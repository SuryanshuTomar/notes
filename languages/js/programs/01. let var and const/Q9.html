Q9: Are var, let, and const block scoped?

let and const are block scoped, meaning they are only accessible within the block they are defined. var is not block scoped; it is function scoped or globally scoped.

{
    var blockVar = 'I am var';
    let blockLet = 'I am let';
    const blockConst = 'I am const';
}
console.log(blockVar); // 'I am var'
// console.log(blockLet); // ReferenceError
// console.log(blockConst); // ReferenceError
